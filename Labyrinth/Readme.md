# Labyrinth (C, Konsole)

**Build:** `make`  
**Run:** `./labyrinth [rows cols ratio]` (z. B. `./labyrinth 12 12 0.2`)  
**Steuerung:** W/A/S/D bewegen, Q beenden.

Projektdateien: `main.c`, `labyrinth.c`, `labyrinth.h`, `Makefile`  
Doku (PDF) liegt in `docs/`.
